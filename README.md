# 💰 FinTrack – Financial Tracker System

FinTrack is a modern **financial tracking web application** built with **React.js** that helps users manage income, expenses, and financial insights in one place. It also provides **real-time market news updates** using a News API.

---

## 🚀 Features

* 📊 **Interactive Dashboard**

  * View balance, total income, and expenses
  * Clean and modern UI

* 💸 **Income & Expense Tracking**

  * Add and manage transactions easily
  * Categorize expenses

* 📅 **Transaction History**

  * Track all past financial activities

* 📈 **Statistics & Insights**

  * Weekly, Monthly, Yearly expense tracking
  * Highest income & expense overview

* 📰 **Real-Time Market News**

  * Integrated News API for finance updates
  * Stay updated with market trends

* 🔔 **Smart Tracking**

  * Understand spending patterns
  * Better financial decisions

* 🌙 **Dark Mode**

  * Toggle between light and dark themes

---

## 🛠️ Tech Stack

* **Frontend:** React.js, HTML, CSS, JavaScript
* **Build Tool:** Vite
* **API:** News API

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/fintrack.git
cd fintrack
```

### 2️⃣ Install dependencies

```bash
npm install
```

### 3️⃣ Add environment variables

Create a `.env` file in the root directory:

```env
VITE_NEWS_API_KEY=your_api_key_here
```

---

### 4️⃣ Run the project

```bash
npm run dev
```

App will run on:

```
http://localhost:5173/
```
